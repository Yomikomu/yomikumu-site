Download Yomikomu

Get started with the Yomikomu manga reader. Choose your preferred installation method below.

Direct JAR Download

Download the JAR file directly and run it on any system with Java installed.

  • Download the latest JAR file from the GitHub releases page
  • Ensure you have Java 8 or higher installed
  • Run with: java -jar Yomikomu-{version}.jar
Download from GitHub Releases

Linux (AUR)

For Arch Linux and Manjaro users, install Yomikomu from the Arch User Repository.

Use your favorite AUR helper or compile from source:

# Using yay (AUR helper) yay -S Yomikomu-reader

Build from Source

For developers or advanced users who want to build Yomikomu from source code.

  • Clone the repository
  • Ensure you have Java Development Kit (JDK) installed
  • Use Apache Maven to build the project
# Clone the repository git clone https://github.com/Yomikomu-Reader/Yomikomu.git cd Yomikomu # Run with Maven mvn clean install; mvn exec:java

System Requirements

  • Operating System: Windows, macOS, or Linux
  • Java: OpenJDK 8 or higher / Oracle Java 8 or higher
  • Memory: Minimum 2GB RAM (4GB recommended)
  • Storage: At least 200MB of free disk space (caching)

For Linux installations via AUR or Flatpak, the package manager will handle Java dependencies automatically.

Yomikomu is an open-source project. View the source on GitHub.
